# Web-Verse-Assignment-1

*Grading System Using Functions & Conditionals*

Objective: Strengthen logic with if/elif/else inside a function.
Task:
Create a function get_grade(score) that:
	•Takes a number (0–100).
	•Returns A/B/C/D/F using conditional logic.

Then:
	•Accept a score list from user (comma-separated).
	•Loop through the list and print each student’s grade using the function.
